让Monaco editor补全python代码

web 端的 python 编辑器起因今年年初的时候接到一个关于量化交易的项目，需要为页面提供支持 python 自动补全、高亮等功能的代码编辑器。目前能找到的编辑器相关方案不算多，在调研阶段我找到的有以下几个： Ace editor Monaco editor 这两个编辑器的特点基本也没啥，ace 诞生时间相对较长（老前辈），而 monaco 作为 vscode 的前身，在 UI 风格上跟 vscode 基本保持一致。从需求的方面来说，ace 其实够用，如果你用的是 react 开发项目，还有 react-ace 将 ace editor 封装成方便易用的 react 组件，ace 本身支持的高亮/补全的语言也相当全面，稍加配置就能满足我的项目需求，而反观 monaco，内置多个语言的高亮，但是补全呢？只有 html、css 以及 typescript。 选用 ace，对于我的项目来说，这个事情就到此为止了，继续埋头搬砖，今天还是加班。 但是这样就不酷了，我想让 Monaco 也能做代码高亮，像 vscode 一样写 python。 开搞经过一番搜索，我大概搞懂了微软对 Mo
